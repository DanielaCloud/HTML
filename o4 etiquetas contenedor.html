🤠Etiquetas contenedor

Las etiquetas div y span en HTML sin elementos no semanticos, lo que significa que no aportan ningún significado especifico sobre su contenido.

Se utilizan principalmente como contenedores generales para agrupar elementos, aplicar estilos con CSS o funcionalidades con JS
